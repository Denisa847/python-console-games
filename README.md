# 🎮 Python Console Games

This repository contains a collection of **small console-based games written in Python**.

Most of these projects were created while learning Python programming, during my **first year of studying computer science**. The goal of these projects was to practice fundamental programming concepts and improve problem-solving skills.

---

## 🧠 Concepts Practiced

These projects focus on practicing core Python concepts such as:

- Object-Oriented Programming
- Game logic and grid-based systems
- Input handling in the terminal
- Basic AI or decision logic
- Code organization across multiple modules
- Simple unit testing
